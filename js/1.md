### 第一题

难易程度：⭐

请确认下列 js 代码运行后，输出的结果是什么

```js
console.log(undefined === null);
console.log(true == 1);
console.log(() => {
  console.log("hello, word") == "object";
});
console.log(Object.is(NaN) === Object.is(NaN));
console.log(Object.is(NaN) == Object.is(NaN));
console.log("a" > "ab");
```

### 答案：

```js
false
true
() => { console.log('hello, word') == 'object'}
true
true
false
```

### 解析：

本题主要考察了一些简单的类型转换问题。
